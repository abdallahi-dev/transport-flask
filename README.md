# 🚍 Projet Gestion Transport Flask

## 📌 Description
Application web de gestion de transport public développée avec Flask et MySQL.

---

## ⚙️ Installation

1. Créer une base de données nommée :
   dbtest

2. Importer le fichier :
   dbtest.sql

3. Installer les dépendances :
   pip install flask flask-mysqldb

4. Lancer l'application :
   python app.py

---

## 🧩 Fonctionnalités

- CRUD Bus
- CRUD Chauffeur
- Gestion des lignes

## 🛠️ Technologies

- Python (Flask)
